# support-ticket-triage-system
Built an AI-based support ticket triage system that processes customer issues, classifies them into categories, applies risk-based escalation, and generates automated responses.

## 📌 Overview

This project simulates a real-world **customer support automation system**.  
It reads support tickets from a dataset, analyzes them using NLP logic, and produces structured outputs for downstream systems.

## ✨ Features

- 🔍 Ticket classification (login, billing, bug, fraud, etc.)
- 🧠 Domain-aware routing (HackerRank, Claude, Visa)
- ⚠️ Escalation decision logic for sensitive issues
- 💬 Automated response generation
- 📊 CSV output generation
- 📝 Logging for traceability (chat transcript)
